# 👜 BagHaven – Full Stack E-commerce Website

BagHaven is a full-stack e-commerce web application built with the MERN Stack. It allows users to browse products, manage shopping carts, and experience a modern online shopping interface.

## 🚀 Features

* User Authentication
* Product Listing
* Product Details
* Wishlist
* Shopping Cart
* Admin Dashboard
* CRUD Operations
* Responsive Design
* REST API Integration

## 🛠️ Tech Stack

### Frontend

* React.js
* JavaScript
* HTML5
* TailwindCSS

### Backend

* Node.js
* Express.js

### Database

* MongoDB

### Tools

* Git
* GitHub
* Postman
